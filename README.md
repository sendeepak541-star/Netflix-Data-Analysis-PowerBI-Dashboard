# 🎬 Netflix Data Analysis Dashboard | Power BI

## 📌 Project Overview
This project focuses on analyzing Netflix content data using Microsoft Power BI. 
The dashboard provides interactive insights into Netflix movies and shows based on views, ratings, genres, countries, and release years.

## 🛠️ Tools & Technologies Used
- Microsoft Power BI
- Power Query (Data Cleaning & Transformation)
- Data Visualization
- DAX Measures

## 📊 Dashboard Insights
- Total Views Analysis
- Highest Viewed Content
- Genre-wise Performance Analysis
- Country-wise View Distribution
- Rating Analysis
- Year-wise Content Analysis
- IMDb Score Analysis

## 📈 Key KPIs
- Total Views: 64.48K
- Highest Views: 249.50M
- Total Genres Analyzed: 8

## 🎯 Business Objective
The objective of this project is to understand content performance trends and identify patterns in viewer engagement across different countries, genres, and ratings.
